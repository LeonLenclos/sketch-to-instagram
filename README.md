# Sketch To Instagram

This is a simple software that allow you to :

1. Draw something

2. Post this drawing on Instagram

## Keys

- `H` to see the Help
- `R` to reset the sketch
- `L` to log in
- `I` to share on instagram
- `ESC` to quit
- `F` to toggle full screen
